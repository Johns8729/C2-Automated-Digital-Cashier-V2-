🏦 TECH BANK RIWI DIGITAL - ATM System

📌 Description

This program simulates the operation of an ATM (Automated Teller Machine) in the console using Python.
It allows a user to authenticate with a PIN and perform basic banking operations.


---

⚙️ Main Features

🔐 User Authentication

Prompts for a username.

Requires a 4-digit PIN.

Maximum of 3 attempts before access is blocked.



---

💰 Available Operations

1. Check Balance

Displays the current account balance.

2. Withdraw Money

Verifies sufficient funds.

Displays date and time of the transaction.

Automatically updates the balance.


3. Deposit Money

Allows the user to add money to the account.

Validates that the amount is positive.

Updates the balance.


4. Exit

Ends the session.



---

📊 Transaction History

Stores all actions performed during the session.

Displays at the end:

User

List of completed operations
